# Climate Change: The Evidence

An interactive educational website exploring the scientific evidence that demonstrates climate change is caused by human activities.

## Project Information

This project was created for **NSCI 204 - Environmental Sciences** at **Metro State University** for the **Fall 2025** semester.

**Author:** Joatan Sampaio  
**Course:** NSCI 204  
**Institution:** Metro State University  
**Semester:** Fall 2025

## Overview

This interactive web-based teaching tool presents ten key points of evidence that demonstrate how we know climate change is caused by human activities.

## Topics Covered

1. The Mechanism: Greenhouse Gases
2. The Spike: 800,000 Years of Data
3. Temperature Follows CO₂
4. The Planet is Reacting
5. Process of Elimination (Natural vs Human causes)
6. The Culprits: Emissions Sources
7. Regional: Minnesota Warming Faster
8. Why It Matters (economy, health, food security, disasters)
9. Scientific Consensus
10. The Path Forward (solutions)

## Usage

Simply open `index.html` in a web browser to view the presentation.
